Which yarn to choose

Hairy yarn

If you want to crochet something warm it's better to use hairy yarn because it holds heat inside.

Basic yarn

If you want to crochet something light for spring or summer or even some stuffed toys it's better to use basic yarn because it's not that big.

Cotton yarn

If you want to make something harder for example a basket it's better to use cotton yarn because it's harder and it can't shape it self that much. Coton yarn is not good for crocheting toys because it is hard and then the toys are hard.
